# Jupyter Notebooks

此目录用于存放策略研究和回测分析的 Jupyter Notebook 文件。

## 推荐使用流程

1. **01_data_download.ipynb** - 下载和保存股票历史数据
2. **02_factor_analysis.ipynb** - 技术指标因子分析
3. **03_strategy_development.ipynb** - 策略开发和信号生成
4. **04_backtest_analysis.ipynb** - 回测结果分析和可视化
5. **05_parameter_optimization.ipynb** - 参数优化和过拟合检验

## 创建新的 Notebook

在开始策略研究时，建议在本目录创建新的 .ipynb 文件。
